# yakata-editor 仕様書

## 概要

グリッドベースの間取り図エディタ。ミステリ小説・TRPG・ゲーム制作などで使う「建物の構造図」を、シンプルな操作で作成するWebアプリケーション。

## ターゲットユーザー

- ミステリ小説の作家（見取り図の作成・トリックの動線検証）
- TRPGのゲームマスター（ダンジョン・建物マップの作成）
- ゲーム制作者（レベルデザインのプロトタイプ）
- 建物の構造を簡単にスケッチしたい人全般

## 設計思想

### やること
- グリッドに沿った四角形（部屋・廊下）の配置
- 部屋のリサイズ・移動・削除・ラベル付け
- 複数選択によるまとめて移動
- PNG / JSON でのエクスポート・インポート
- タッチデバイス（iPad + Apple Pencil）対応

### やらないこと
- 3D表示
- 家具・装飾の配置
- 建築CAD的な精密製図
- リアルタイムコラボレーション（初期段階では）
- ユーザーアカウント・サーバーサイド処理

## 機能一覧

### v1.0（初回リリース）

#### 基本操作
| 操作 | 動作 |
|------|------|
| 空白をドラッグ | 新しい四角形（部屋）を作成 |
| 部屋をクリック | 選択（ハンドル表示） |
| Shift+クリック | 複数選択に追加/解除 |
| 選択中の部屋をドラッグ | 移動（複数選択時はまとめて移動） |
| ハンドルをドラッグ | リサイズ（単一選択時のみ） |
| ダブルクリック | 部屋名の入力 |
| Delete / Backspace | 選択中の部屋を削除 |
| ⌘Z / Ctrl+Z | 元に戻す |

#### グリッド
- すべての操作がグリッドにスナップする
- 5マスごとに太い線（方眼紙スタイル）
- グリッドサイズは固定（20px）、将来的に可変も検討

#### 描画スタイル
- 部屋: 白地に黒い枠線（壁）
- 選択中: 青い枠線 + ハンドル
- 作成中: 青い点線プレビュー
- ラベル: 部屋中央にテキスト表示

#### 保存・読み込み
- **localStorage**: 自動保存（ブラウザを閉じても保持）
- **JSON保存**: ファイルとしてダウンロード
- **JSON読み込み**: ファイルから復元
- **PNG出力**: 画像としてエクスポート（選択状態なしのクリーンな出力）

### v1.1（候補機能）

#### ドア（壁の切れ目）
- 壁をクリックしてドア（隙間）を配置
- 部屋と部屋の接続を表現

#### テキスト注釈
- 部屋以外の場所に自由テキストを配置
- 「防犯カメラ」「階段↑2F」などのメモ用途

#### 矢印・動線
- 2点間に矢印を描画
- 移動ルートや視線の検証用
- 色分け対応（人物ごとに色を変えるなど）

#### 階層管理
- 複数フロア（1F / 2F / 3F）の切り替え
- フロアごとに独立した部屋配置
- フロア間の階段位置の整合性表示

### v2.0（将来構想）

#### タッチデバイス対応
- touchstart / touchmove / touchend イベント対応
- iPad + Apple Pencil での操作
- ピンチズーム

#### コピー・ペースト
- 選択中の部屋を⌘C / ⌘V で複製
- 左右対称の間取りを効率的に作成

#### 範囲選択
- 空白をドラッグで選択範囲（矩形）を描画
- 範囲内の部屋をまとめて選択

#### ズーム
- 拡大・縮小表示
- ピンチズーム（タッチ対応時）

#### テンプレート
- よくある構造（L字型館、コの字型など）のプリセット

## 技術スタック

| 項目 | 技術 |
|------|------|
| 言語 | TypeScript |
| ビルド | Vite |
| 描画 | Canvas API |
| UIフレームワーク | なし（バニラ）or Svelte（UIが複雑化した場合） |
| ホスティング | GitHub Pages |
| CI/CD | GitHub Actions（Vite build → Pages deploy） |

### 技術選定の理由

- **Canvas API**: SVGより多数の矩形描画が高速。この用途ではCanvasが最適
- **TypeScript**: 型安全性。部屋データの構造を型で保証
- **Vite**: 軽量・高速なビルドツール。設定が最小限
- **GitHub Pages**: 静的ファイルのみで完結するため、サーバー不要

## データ構造

### 部屋（Room）

```typescript
interface Room {
  id: string;        // 一意識別子（UUID）
  x: number;         // グリッド座標 X
  y: number;         // グリッド座標 Y
  w: number;         // 幅（グリッド単位）
  h: number;         // 高さ（グリッド単位）
  label: string;     // 部屋名
  floor?: number;    // フロア番号（v1.1〜）
}
```

### プロジェクト（保存ファイル形式）

```typescript
interface Project {
  version: number;       // ファイル形式バージョン
  name: string;          // プロジェクト名
  gridSize: number;      // グリッドサイズ（px）
  rooms: Room[];         // 部屋の配列
  // v1.1〜
  doors?: Door[];        // ドアの配列
  annotations?: Annotation[]; // 注釈の配列
  arrows?: Arrow[];      // 矢印の配列
}
```
