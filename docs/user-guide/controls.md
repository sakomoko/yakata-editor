# 操作リファレンス

## マウス操作

| 操作 | 対象 | 動作 |
|------|------|------|
| ドラッグ | 空白エリア | 新しい部屋を作成 |
| クリック | 部屋 | 部屋を選択 |
| Shift+クリック | 部屋 | 複数選択に追加/解除 |
| ドラッグ | 選択中の部屋 | 部屋を移動（複数選択時・連結時は一括移動） |
| ドラッグ | リサイズハンドル | 部屋のサイズを変更（単一選択時のみ） |
| 右クリック | 部屋の壁 | コンテキストメニューを表示（窓・ドアの追加、連結/連結解除、重なり順序の変更） |
| 右クリック | 窓 | コンテキストメニューを表示（窓の削除） |
| 右クリック | ドア | コンテキストメニューを表示（開き方向の切替・ヒンジ位置の切替・ドアの削除） |
| 右クリック | 開口 | コンテキストメニューを表示（開口の削除） |
| 右クリック | 自動生成された開口 | 「この開口は隣室と連動しています」と表示（削除不可） |
| ドラッグ | 壁の窓 | 窓を壁に沿って移動（別の壁への移動も可能） |
| ドラッグ | ドア（壁線・弧・扇形エリア） | ドアを壁に沿って移動（別の壁への移動も可能） |
| ドラッグ | 開口 | 開口を壁に沿って移動（別の壁への移動も可能） |
| ドラッグ | 壁オブジェクトの端（リサイズハンドル） | 壁オブジェクト（窓・ドア・開口）の幅を伸縮 |
| 右クリック | 部屋の内部 | コンテキストメニューに「直線階段を配置」「折り返し階段を配置」が追加表示 |
| 右クリック | 階段 | コンテキストメニューを表示（方向変更・階段の削除） |
| ドラッグ | 階段 | 部屋内で階段を移動（部屋境界内に制約） |
| ドラッグ | 階段の四隅ハンドル | 階段のサイズを変更 |
| ダブルクリック | 部屋 | 部屋設定ダイアログを表示（部屋名・フォントサイズ編集、Enterキーで確定可能） |
| 右クリック | 部屋の内部 | コンテキストメニュー先頭に「部屋名を変更」が表示（部屋設定ダイアログを開く） |

## キーボードショートカット

| キー | 動作 |
|------|------|
| `Delete` / `Backspace` | アクティブな階段を削除（なければ選択中の部屋を削除） |
| `⌘Z` / `Ctrl+Z` | 元に戻す（Undo） |
| `⌘]` / `Ctrl+]` | 選択中の部屋を1つ前面へ移動 |
| `⌘[` / `Ctrl+[` | 選択中の部屋を1つ背面へ移動 |
| `⌘Shift+]` / `Ctrl+Shift+]` | 選択中の部屋を最前面へ移動 |
| `⌘Shift+[` / `Ctrl+Shift+[` | 選択中の部屋を最背面へ移動 |

## ツールバー

| ボタン | 動作 |
|--------|------|
| 新規 | 新しいプロジェクトを作成 |
| 保存 | JSONファイルとしてダウンロード |
| 読込 | JSONファイルから復元 |
| PNG | PNG画像としてエクスポート |
| 元に戻す | 直前の操作を取り消し |

## グリッド

- すべての操作はグリッド（20px単位）にスナップします
- 5マスごとに太い線が引かれます（方眼紙スタイル）
- キャンバスは無限に広がります（任意の位置に部屋を配置可能）

## 描画スタイル

| 状態 | 見た目 |
|------|--------|
| 通常の部屋 | 白地に黒い枠線 |
| 選択中の部屋 | 青い枠線 + 8つのリサイズハンドル |
| 作成中（ドラッグ中） | 青い点線のプレビュー |
| 部屋名 | 部屋の中央にテキスト表示 |
| 窓 | 壁の切れ目（二重平行線） |
| ドア | ヒンジ点からのパネル線 + 四分円弧 |
| 開口 | 壁の切れ目（隙間のみ、シンボルなし） |
| 自動生成された開口 | 隣接部屋の壁オブジェクトと連動する開口（移動・リサイズ・削除不可） |
| ドラッグ中の窓・ドア | オレンジ色の太いライン |
| 直線階段 | 矩形 + 踏み板線（半グリッド間隔） + 方向矢印 |
| 折り返し階段 | 矩形 + 中央仕切り線 + 両側踏み板線 + U字型方向矢印 |
| アクティブな階段 | オレンジ色の太いライン |
| リサイズハンドル（階段） | オレンジ色の正方形マーカー（選択中の部屋のみ表示） |
| リサイズハンドル（壁オブジェクト） | オレンジ色のひし形マーカー（選択中の部屋のみ表示） |
