# 操作リファレンス

## マウス操作

| 操作 | 対象 | 動作 |
|------|------|------|
| ドラッグ | 空白エリア | 新しい部屋を作成（既存の部屋を覆った場合は範囲選択） |
| Shift+ドラッグ | 空白エリア | 範囲選択で既存の選択に追加 |
| クリック | 部屋 | 部屋を選択 |
| Shift+クリック | 部屋 | 複数選択に追加（最後の1部屋は解除不可、Shift+ドラッグでサブグリッド移動するため） |
| ドラッグ | 選択中の部屋 | 部屋を移動（複数選択時・連結時は一括移動） |
| ドラッグ | リサイズハンドル | 部屋のサイズを変更（単一選択時のみ） |
| ドラッグ | グループBBの4隅ハンドル | 選択中の複数部屋をアスペクト比維持で一括スケーリング（2つ以上選択時） |
| 右クリック | 部屋の壁 | コンテキストメニューを表示（窓・ドアの追加、連結/連結解除、重なり順序の変更） |
| 右クリック | 窓 | コンテキストメニューを表示（窓の削除） |
| 右クリック | ドア | コンテキストメニューを表示（開き方向の切替・ヒンジ位置の切替・ドアの削除） |
| 右クリック | 開口 | コンテキストメニューを表示（開口の削除） |
| 右クリック | 自動生成された開口 | 「この開口は隣室と連動しています」と表示（削除不可） |
| ドラッグ | 壁の窓 | 窓を壁に沿って移動（別の壁への移動も可能） |
| ドラッグ | ドア（壁線・弧・扇形エリア） | ドアを壁に沿って移動（別の壁への移動も可能） |
| ドラッグ | 開口 | 開口を壁に沿って移動（別の壁への移動も可能） |
| ドラッグ | 壁オブジェクトの端（リサイズハンドル） | 壁オブジェクト（窓・ドア・開口）の幅を伸縮 |
| 右クリック | 部屋の内部 | コンテキストメニューに「四角形に変換」「直線階段を配置」「折り返し階段を配置」「死体を配置」「マーカーを配置」「テキストを配置」「防犯カメラを配置」が表示 |
| 右クリック | 四角形に変換済みの部屋 | コンテキストメニューに「矩形に戻す」が表示（「四角形に変換」の代わり） |
| ドラッグ | 四角形部屋の頂点ハンドル（丸/青） | 頂点をドラッグして角度を自由に変更（他部屋の頂点・辺にスナップ、グリッドスナップなし） |
| 右クリック | 階段 | コンテキストメニューを表示（方向変更・階段の削除） |
| 右クリック | マーカー | コンテキストメニューを表示（ラベル変更・向き変更（死体のみ）・マーカーの削除） |
| 右クリック | 防犯カメラ | コンテキストメニューを表示（色プリセット変更（青・赤・緑・黄）・カメラの削除） |
| ドラッグ | カメラの回転ハンドル（丸/オレンジ） | カメラの向きを変更 |
| ドラッグ | カメラの距離ハンドル（四角/緑） | 視野の到達距離を調整 |
| ドラッグ | カメラの広がりハンドル（ダイヤ/青） | 視野角を調整 |
| ドラッグ | 階段 | 階段を移動（グリッドスナップ、別の部屋にドロップで移し替え、部屋外でドロップすると元の部屋に戻る） |
| ドラッグ | マーカー | マーカーを自由に移動（グリッドに縛られない、別の部屋にドロップで移し替え、部屋外でドロップすると元の部屋に戻る） |
| ドラッグ | 階段の四隅ハンドル | 階段のサイズを変更（グリッドスナップ） |
| ドラッグ | マーカーの四隅ハンドル | マーカーのサイズを自由に変更（ラベルのフォントサイズが追従） |
| ダブルクリック | マーカー | マーカー設定ダイアログを表示（ラベル編集・フォントサイズ変更、Enterキーで確定可能） |
| ダブルクリック | 部屋 | 部屋設定ダイアログを表示（部屋名・フォントサイズ編集、Enterキーで確定可能） |
| 右クリック | 空白エリア | コンテキストメニューに「テキストを配置」が表示（自由配置テキストの作成） |
| 右クリック | 自由配置テキスト | コンテキストメニューを表示（テキストを変更・最前面/最背面切替・テキストを削除） |
| ドラッグ | 自由配置テキスト | テキストを移動（グリッドスナップ） |
| ドラッグ | 自由配置テキストの四隅ハンドル | テキストのサイズを変更 |
| ダブルクリック | 自由配置テキスト | テキスト設定ダイアログを表示（テキスト・フォントサイズ編集） |
| 右クリック | 部屋の内部 | コンテキストメニュー先頭に「部屋名を変更」が表示（部屋設定ダイアログを開く） |
| クリック | フリーペイントストローク | ストロークを選択（通常モード時） |
| ドラッグ | 選択中のストローク | ストロークを移動（通常モード時） |
| 右クリック | フリーペイントストローク | コンテキストメニューを表示（色変更・太さ変更・削除） |

## キーボードショートカット

| キー | 動作 |
|------|------|
| `?` | ショートカットキー一覧ダイアログの表示/非表示 |
| `P` | ペイントモードの切替（ON/OFF） |
| `Delete` / `Backspace` | アクティブなFreeText・階段・マーカー・カメラを削除（なければ選択中の部屋・FreeText・FreeStrokeを削除） |
| `⌘Z` / `Ctrl+Z` | 元に戻す（Undo） |
| `⌘]` / `Ctrl+]` | 選択中の部屋を1つ前面へ移動 |
| `⌘[` / `Ctrl+[` | 選択中の部屋を1つ背面へ移動 |
| `⌘Shift+]` / `Ctrl+Shift+]` | 選択中の部屋を最前面へ移動 |
| `⌘Shift+[` / `Ctrl+Shift+[` | 選択中の部屋を最背面へ移動 |

## タブバー

| 操作 | 動作 |
|------|------|
| タブをクリック | プロジェクトを切り替え |
| タブをダブルクリック | プロジェクト名をインライン編集 |
| タブを右クリック | コンテキストメニューを表示（プロジェクトを複製・名前を変更・タブを閉じる） |
| タブの「×」をクリック | タブを閉じる（データは残る、最後の1つは閉じ不可） |
| 「+」ボタン | 新規プロジェクトをタブで追加 |
| 「☰」ボタン | プロジェクト一覧モーダルを開く |

## プロジェクト一覧モーダル

| 操作 | 動作 |
|------|------|
| 「開く」ボタン | プロジェクトを新しいタブで開く（既に開いている場合はそのタブに切り替え） |
| 「複製」ボタン | プロジェクトを複製して新しいタブで開く（「〇〇 のコピー」という名前で作成） |
| 🗑 ボタン | プロジェクトを削除（最後の1つは削除不可） |

## フッターバー（画面下部）

画面下部のバーにはステータス表示とコマンドボタンが1行に統合されています。左側にステータス（座標・部屋数・ズーム率）、右側にコマンドボタンが配置されます。

| ボタン | 動作 |
|--------|------|
| 新規プロジェクト | 新しいプロジェクトをタブで追加 |
| 保存 | JSONファイルとしてダウンロード |
| 開く | JSONファイルから復元 |
| PNG出力 | PNG画像としてエクスポート |
| 戻す (⌘Z / Ctrl+Z) | 直前の操作を取り消し |
| ペン (P) | ペイントモードの切替 |
| ? | ショートカットキー一覧ダイアログを表示（右端に配置） |

## ペイントモード

`P` キー、またはツールバーの「ペン (P)」ボタンでペイントモードに切り替えられます。

| 操作 | 動作 |
|------|------|
| ドラッグ | フリーハンドでストロークを描画（ピクセル単位、グリッドスナップなし） |
| Shift+ドラッグ | 水平・垂直・45度方向に制約された直線を描画 |

ペイントモードON時はツールバーに描画設定が表示されます:

| 設定 | 説明 |
|------|------|
| 色ピッカー | ストロークの色を選択 |
| 太さセレクタ | ストロークの太さを選択（1/2/3/6/10px） |
| 不透明度セレクタ | ストロークの不透明度を選択（30/50/80/100%） |

ペイントモード中はダブルクリック・右クリックメニューは無効化されます。ステータスバーに「ペイントモード」と表示されます。

## グリッド

- すべての操作はグリッド（20px単位）にスナップします
- Shiftキーを押しながら操作するとグリッドスナップが無効になり、サブグリッド（ピクセル単位）で配置できます
  - 対象: 部屋の作成・移動・リサイズ、壁オブジェクトの移動・リサイズ、自由配置テキストの移動・リサイズ
  - 非対象: グループリサイズ、インテリアオブジェクト（階段・マーカー）、フリーストローク
  - Shiftを離すと自動的にグリッドに揃います
- 5マスごとに太い線が引かれます（方眼紙スタイル）
- キャンバスは無限に広がります（任意の位置に部屋を配置可能）

## 描画スタイル

| 状態 | 見た目 |
|------|--------|
| 通常の部屋 | 白地に黒い枠線 |
| 選択中の部屋 | 青い枠線 + 8つのリサイズハンドル（単一選択時、矩形のみ） |
| 四角形部屋 | 4頂点を結ぶ形状で描画（斜め壁対応） |
| 選択中の四角形部屋 | 青い枠線 + 4つの頂点ハンドル（丸/青） |
| 複数選択中の部屋群 | 青い破線のバウンディングボックス + 4隅のリサイズハンドル |
| 作成中（ドラッグ中） | 青い点線のプレビュー |
| 範囲選択中（ドラッグ中） | 青い半透明矩形 + 破線ボーダー + 包含部屋のハイライト |
| 部屋名 | 部屋の中央にテキスト表示 |
| 窓 | 壁の切れ目（二重平行線） |
| ドア | ヒンジ点からのパネル線 + 四分円弧 |
| 開口 | 壁の切れ目（隙間のみ、シンボルなし） |
| 自動生成された開口 | 隣接部屋の壁オブジェクトと連動する開口（移動・リサイズ・削除不可） |
| ドラッグ中の窓・ドア | オレンジ色の太いライン |
| 直線階段 | 矩形 + 踏み板線（半グリッド間隔） + 方向矢印 |
| 折り返し階段 | 矩形 + 中央仕切り線 + 両側踏み板線 + U字型方向矢印 |
| 死体マーカー | チョークアウトライン風の棒人間シルエット（囲み線なし） |
| ピンマーカー | ピンアイコン + ラベルテキスト（囲み線なし） |
| テキストマーカー | ラベルテキストのみ（囲み線なし） |
| アクティブなマーカー | オレンジ色の太いライン |
| アクティブな階段 | オレンジ色の太いライン |
| 防犯カメラ | 四角い本体 + レンズ台形のアイコン |
| カメラの視野コーン | 半透明の扇形（デフォルト青、部屋の外まで描画） |
| カメラのハンドル | 回転（丸/オレンジ）、距離（四角/緑）、広がり（ダイヤ/青×2） |
| リサイズハンドル（階段） | オレンジ色の正方形マーカー（選択中の部屋のみ表示） |
| リサイズハンドル（壁オブジェクト） | オレンジ色のひし形マーカー（選択中の部屋のみ表示） |
| 頂点スナップインジケーター | オレンジ色の丸（他部屋の頂点にスナップ中） |
| 辺スナップインジケーター | 緑色の丸 + 十字（他部屋の辺上にスナップ中） |
| 自由配置テキスト（front層） | 部屋の上にテキスト表示 |
| 自由配置テキスト（back層） | 部屋の下にテキスト表示 |
| 選択中の自由配置テキスト | 青い破線の枠線 |
| アクティブな自由配置テキスト | オレンジ色の破線枠線 + 四隅にリサイズハンドル |
| フリーペイントストローク | 指定色・太さ・不透明度でフリーハンド描画（常に最前面） |
| 選択中のフリーペイントストローク | 青い破線のバウンディングボックス |
