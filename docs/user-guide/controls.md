# 操作リファレンス

## マウス操作

| 操作 | 対象 | 動作 |
|------|------|------|
| ドラッグ | 空白エリア | 新しい部屋を作成 |
| クリック | 部屋 | 部屋を選択 |
| Shift+クリック | 部屋 | 複数選択に追加/解除 |
| ドラッグ | 選択中の部屋 | 部屋を移動（複数選択時は一括移動） |
| ドラッグ | リサイズハンドル | 部屋のサイズを変更（単一選択時のみ） |
| 右クリック | 部屋の壁 | コンテキストメニューを表示（窓の追加・削除） |
| ドラッグ | 壁の窓 | 窓を壁に沿って移動 |
| ダブルクリック | 部屋 | 部屋名を編集 |

## キーボードショートカット

| キー | 動作 |
|------|------|
| `Delete` / `Backspace` | 選択中の部屋を削除 |
| `⌘Z` / `Ctrl+Z` | 元に戻す（Undo） |

## ツールバー

| ボタン | 動作 |
|--------|------|
| 新規 | 新しいプロジェクトを作成 |
| 保存 | JSONファイルとしてダウンロード |
| 読込 | JSONファイルから復元 |
| PNG | PNG画像としてエクスポート |
| 元に戻す | 直前の操作を取り消し |

## グリッド

- すべての操作はグリッド（20px単位）にスナップします
- 5マスごとに太い線が引かれます（方眼紙スタイル）
- キャンバスサイズ: 100×75グリッド（2000×1500px）

## 描画スタイル

| 状態 | 見た目 |
|------|--------|
| 通常の部屋 | 白地に黒い枠線 |
| 選択中の部屋 | 青い枠線 + 8つのリサイズハンドル |
| 作成中（ドラッグ中） | 青い点線のプレビュー |
| 部屋名 | 部屋の中央にテキスト表示 |
| 窓 | 壁の切れ目（水色のライン） |
