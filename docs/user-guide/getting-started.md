# はじめに

## yakata-editorとは

グリッドベースの間取り図エディタです。ミステリ小説の見取り図、TRPGのダンジョンマップ、ゲームのレベルデザインなど、建物の構造図をシンプルな操作で作成できます。

## 起動方法

ブラウザでアプリケーションを開くと、グリッド（方眼紙）が表示されたキャンバスが表示されます。画面上部にヘッダーとタブバー、下部にステータスバーとツールバーが表示されます。

## 基本的な使い方

### 部屋を作る

キャンバスの空白部分をドラッグすると、新しい部屋（四角形）が作成されます。ドラッグ中は青い点線でプレビューが表示されます。

### 部屋に名前をつける

部屋をダブルクリックすると、名前を入力するダイアログが表示されます。入力した名前は部屋の中央に表示されます。

### 部屋を選択する

部屋をクリックすると選択され、青い枠線とリサイズハンドルが表示されます。

### 部屋を移動する

選択中の部屋をドラッグすると移動できます。すべての移動はグリッドにスナップします。

### 部屋のサイズを変更する

部屋を選択すると表示される8つのハンドル（角と辺の中央）をドラッグしてリサイズできます。

### 部屋を削除する

部屋を選択した状態で `Delete` キーまたは `Backspace` キーを押すと削除されます。

### 壁に窓を配置する

部屋の壁を右クリックすると、コンテキストメニューが表示されます。「窓を追加」を選ぶと、クリックした位置に窓が配置されます。窓はドラッグで壁に沿って移動できます。窓を右クリックして「窓を削除」で取り除けます。

### フリーペイントで描き込む

`P` キーまたはツールバーの「ペン (P)」ボタンでペイントモードに切り替えると、間取り図の上にフリーハンドで線を描けます。犯人の逃走経路やメモの書き込みに便利です。ペイントモード中はツールバーに色・太さ・不透明度の設定が表示されます。もう一度 `P` キーを押すと通常モードに戻ります。

### 操作を元に戻す

`⌘Z`（Mac）または `Ctrl+Z`（Windows/Linux）で直前の操作を元に戻せます。ツールバーの「元に戻す」ボタンでも同様です。

## プロジェクトの管理

### 複数プロジェクト

タブバーで複数の間取り図プロジェクトを同時に管理できます。

- **新しいプロジェクト**: タブバーの「+」ボタン、またはツールバーの「新規プロジェクト」ボタンで追加
- **タブの切り替え**: タブをクリックで切り替え。各プロジェクトのUndoヒストリ・ズーム・パン位置は独立して保持されます
- **プロジェクト名の変更**: タブをダブルクリックで名前を編集
- **タブを閉じる**: タブの「×」ボタンで閉じます（データはlocalStorageに残ります）
- **プロジェクト一覧**: タブバー右端の「☰」ボタンで一覧モーダルを開き、閉じたプロジェクトを再度開いたり、不要なプロジェクトを削除できます

### 保存と読み込み

| ツールバーボタン | 機能 |
|--------------|------|
| 新規プロジェクト | 新しいプロジェクトをタブで追加 |
| 保存 | 現在のプロジェクトをJSONファイルとしてダウンロード |
| 開く | 保存したJSONファイルを読み込み |
| PNG出力 | 画像としてエクスポート |

作業中の内容はブラウザのlocalStorageに自動保存されるため、ブラウザを閉じても次回アクセス時に復元されます。
