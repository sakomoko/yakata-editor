# はじめに

## yakata-editorとは

グリッドベースの間取り図エディタです。ミステリ小説の見取り図、TRPGのダンジョンマップ、ゲームのレベルデザインなど、建物の構造図をシンプルな操作で作成できます。

## 起動方法

ブラウザでアプリケーションを開くと、グリッド（方眼紙）が表示されたキャンバスが表示されます。上部のツールバーと下部のステータスバーが利用できます。

## 基本的な使い方

### 部屋を作る

キャンバスの空白部分をドラッグすると、新しい部屋（四角形）が作成されます。ドラッグ中は青い点線でプレビューが表示されます。

### 部屋に名前をつける

部屋をダブルクリックすると、名前を入力するダイアログが表示されます。入力した名前は部屋の中央に表示されます。

### 部屋を選択する

部屋をクリックすると選択され、青い枠線とリサイズハンドルが表示されます。

### 部屋を移動する

選択中の部屋をドラッグすると移動できます。すべての移動はグリッドにスナップします。

### 部屋のサイズを変更する

部屋を選択すると表示される8つのハンドル（角と辺の中央）をドラッグしてリサイズできます。

### 部屋を削除する

部屋を選択した状態で `Delete` キーまたは `Backspace` キーを押すと削除されます。

### 操作を元に戻す

`⌘Z`（Mac）または `Ctrl+Z`（Windows/Linux）で直前の操作を元に戻せます。ツールバーの「元に戻す」ボタンでも同様です。

## 保存と読み込み

| ツールバーボタン | 機能 |
|--------------|------|
| 新規 | 新しいプロジェクトを作成（現在の内容はクリアされます） |
| 保存 | JSONファイルとしてダウンロード |
| 読込 | 保存したJSONファイルを読み込み |
| PNG | 画像としてエクスポート |

作業中の内容はブラウザのlocalStorageに自動保存されるため、ブラウザを閉じても次回アクセス時に復元されます。
