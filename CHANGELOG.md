# Changelog

このプロジェクトの注目すべき変更はすべてこのファイルに記録されます。
フォーマットは [Keep a Changelog](https://keepachangelog.com/ja/1.1.0/) に準拠しています。

## [Unreleased]

### Added

- 部屋の重なり順序（Z-Order）を変更できる機能
  - 右クリックメニューから「最前面へ移動」「1つ前面へ移動」「1つ背面へ移動」「最背面へ移動」が可能
  - キーボードショートカット: `⌘]/Ctrl+]`（1つ前面）、`⌘[/Ctrl+[`（1つ背面）、`+Shift`で最前面/最背面
  - コンテキストメニューにセパレータ（区切り線）表示を追加
- 部屋の壁にドアを配置できる機能
  - 壁の右クリックでコンテキストメニューから「ドアを配置」が可能
  - 内開き/外開きの切替に対応（右クリック→「開き方向を切替」）
  - ヒンジ点から弧を描く間取り図標準のビジュアル表示
  - ドアの扇形エリア全体をクリックしてドラッグ移動・右クリックメニュー操作が可能
  - ドアの位置・開き方向は保存・復元に対応
- 部屋の壁に窓を配置できる機能
  - 壁の右クリックでコンテキストメニューを表示し、窓の追加・削除が可能
  - 窓は壁上でドラッグ移動可能
  - 部屋のリサイズ時に窓の位置を自動クランプ
  - 窓の位置は保存・復元に対応
- 窓のドラッグ中に別の壁へ移動できる機能
  - ドラッグ中にマウスに最も近い壁へ窓を自動的に移動
  - ドラッグ中の窓をオレンジ色でハイライト表示（部屋全体の選択表示を抑制）
  - ホバー時にgrabカーソル、ドラッグ中にgrabbingカーソルを表示

## [0.1.0] - 2026-03-13

### Added

- グリッドベースの間取り図エディタの基本機能
  - 部屋の作成（ドラッグで矩形描画）
  - 部屋の選択・複数選択（Shift+クリック）
  - 部屋の移動・リサイズ（8方向ハンドル）
  - 部屋の削除（Delete/Backspace）
  - 部屋名のラベル編集（ダブルクリック）
- Undo機能（⌘Z / Ctrl+Z、最大50件）
- 保存・読み込み機能
  - localStorage自動保存
  - JSONファイルのエクスポート/インポート
  - PNG画像エクスポート
- Vite + TypeScript開発環境
- ESLint + Prettier + Vitestによるコード品質管理
