# Changelog

このプロジェクトの注目すべき変更はすべてこのファイルに記録されます。
フォーマットは [Keep a Changelog](https://keepachangelog.com/ja/1.1.0/) に準拠しています。

## [Unreleased]

### Added

- ドラッグ範囲選択機能
  - 空白エリアからドラッグして既存の部屋を覆うと、部屋作成ではなく範囲選択として動作
  - ドラッグ矩形に完全に包含される部屋のみが選択対象
  - ドラッグ中にリアルタイムでプレビュー表示（青い半透明矩形 + 包含部屋のハイライト）
  - Shift+範囲選択で既存の選択に追加
- 部屋設定ダイアログでEnterキーによる確定をサポート（IME変換中は除外）
- 部屋の右クリックコンテキストメニューに「部屋名を変更」メニューを追加
- ドアのヒンジ位置を変更できる機能
  - 右クリックメニューの「ヒンジ位置を切替」でヒンジをstart側/end側に切替可能
  - 既存データはデフォルトstart（従来動作）で後方互換性を維持
- 部屋内に階段を配置できる機能（初の部屋内オブジェクト）
  - 直線階段（デフォルト2×3グリッド）と折り返し階段（デフォルト4×3グリッド）の2種類
  - 方向は4方向（N/E/S/W）で、矢印で昇降方向を表示
  - 右クリックメニューから配置・方向変更・削除が可能
  - ドラッグで移動、四隅ハンドルでリサイズ（部屋境界内に制約）
  - Delete/Backspaceキーでアクティブな階段を削除
  - 部屋リサイズ時に階段を自動クランプ
  - localStorage/JSONファイルでの保存・復元に対応

### Fixed

- 隣接壁に配置した開口がクリック後に消える問題を修正（autoGeneratedフラグで自動生成開口を正確に区別）
- 隣接壁上の壁オブジェクト（窓・ドア・開口）が移動・リサイズできない問題を修正（ヒットテストで自動生成開口をスキップ）
- 壁オブジェクトのドラッグ中に隣室の壁が透けて操作状況がわかりにくい問題を修正（ドラッグ中もペア開口をリアルタイム同期）
- 狭い壁（2グリッド幅）で開口のリサイズハンドルが部屋リサイズハンドルに隠されて掴めない問題を修正
- 外開きドア（outward swing）の弧・パネル線が隣接部屋の塗りつぶしで隠される問題を修正
  - 全部屋描画後に外開きドアのみを上書き描画する2パス方式を導入

### Changed

- キャンバスを無限キャンバスに変更（固定2000×1500pxの境界を撤廃）
  - グリッドが全方向に無限に広がり、任意の位置（負の座標含む）に部屋を配置可能
  - PNGエクスポートは全部屋のバウンディングボックスから動的にサイズを決定

### Added

- 隣接部屋の壁オブジェクト自動同期機能
  - 壁オブジェクト（ドア・窓・開口）を配置すると、隣接する部屋の共有壁に開口を自動作成
  - 壁オブジェクトの移動・リサイズ・削除に連動してペア開口も更新・削除
  - 部屋の移動・リサイズ・削除時に全ペア開口を自動的に再構築
  - 自動生成された開口はドラッグ移動・リサイズ・削除が不可（コンテキストメニューに案内表示）
  - 壁オブジェクトが共有壁の範囲を超える場合は開口の幅を自動クランプ
- 壁オブジェクト（窓・ドア・開口）の伸縮リサイズ機能
  - 開口部の端をドラッグして幅を変更可能（グリッドスナップ対応）
  - 壁境界を超えない制約と他オブジェクトとのオーバーラップ防止
  - 部屋選択時にオレンジ色のひし形リサイズハンドルを表示
  - 端にホバーするとリサイズカーソル（ew-resize/ns-resize）を表示
- 部屋の連結機能
  - 隣接する部屋同士を連結し、ドラッグ移動時にまとめて動かせる
  - 2つ以上の部屋を選択 → 右クリック →「連結」で連結を作成
  - 連結済みの部屋を選択 → 右クリック →「連結解除」で解除
  - 連結はUndoや保存/読込に自動対応
- 壁に開口（Opening）を配置できる機能
  - 壁の右クリックでコンテキストメニューから「開口を配置」が可能
  - 壁線が途切れるだけのシンプルな表現（ドアの弧や窓の二重線なし）
  - ホールと通路の連絡部分などに使用
  - 開口のドラッグ移動・削除・保存/復元に対応
- 部屋の重なり順序（Z-Order）を変更できる機能
  - 右クリックメニューから「最前面へ移動」「1つ前面へ移動」「1つ背面へ移動」「最背面へ移動」が可能
  - キーボードショートカット: `⌘]/Ctrl+]`（1つ前面）、`⌘[/Ctrl+[`（1つ背面）、`+Shift`で最前面/最背面
  - コンテキストメニューにセパレータ（区切り線）表示を追加
- 部屋の壁にドアを配置できる機能
  - 壁の右クリックでコンテキストメニューから「ドアを配置」が可能
  - 内開き/外開きの切替に対応（右クリック→「開き方向を切替」）
  - ヒンジ点から弧を描く間取り図標準のビジュアル表示
  - ドアの扇形エリア全体をクリックしてドラッグ移動・右クリックメニュー操作が可能
  - ドアの位置・開き方向は保存・復元に対応
- 部屋の壁に窓を配置できる機能
  - 壁の右クリックでコンテキストメニューを表示し、窓の追加・削除が可能
  - 窓は壁上でドラッグ移動可能
  - 部屋のリサイズ時に窓の位置を自動クランプ
  - 窓の位置は保存・復元に対応
- 窓のドラッグ中に別の壁へ移動できる機能
  - ドラッグ中にマウスに最も近い壁へ窓を自動的に移動
  - ドラッグ中の窓をオレンジ色でハイライト表示（部屋全体の選択表示を抑制）
  - ホバー時にgrabカーソル、ドラッグ中にgrabbingカーソルを表示

## [0.1.0] - 2026-03-13

### Added

- グリッドベースの間取り図エディタの基本機能
  - 部屋の作成（ドラッグで矩形描画）
  - 部屋の選択・複数選択（Shift+クリック）
  - 部屋の移動・リサイズ（8方向ハンドル）
  - 部屋の削除（Delete/Backspace）
  - 部屋名のラベル編集（ダブルクリック）
- Undo機能（⌘Z / Ctrl+Z、最大50件）
- 保存・読み込み機能
  - localStorage自動保存
  - JSONファイルのエクスポート/インポート
  - PNG画像エクスポート
- Vite + TypeScript開発環境
- ESLint + Prettier + Vitestによるコード品質管理
