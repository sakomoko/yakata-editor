# yakata-editor（館エディタ）

グリッドベースの間取り図エディタ。推理作家・TRPGゲームマスター・ゲーム開発者向けのWebアプリケーション。

**https://sakomoko.github.io/yakata-editor/**

## 特徴

- **グリッドスナップ** — すべての操作がグリッドに沿い、きれいな間取り図が描ける
- **四角形 / 自由四角形の部屋** — 通常の矩形に加え、頂点を個別にドラッグして台形・平行四辺形などの変形部屋も作成可能
- **壁オブジェクト** — ドア（開き戸）・窓・開口部を壁上に配置。隣接する部屋間で自動同期
- **インテリアオブジェクト** — 階段・折りたたみ階段・マーカー・テキストラベルを部屋内に配置
- **防犯カメラ** — 視野角(FOV)付きのカメラアイコンを配置。色分けでカメラグループを識別
- **フリーペイント** — ペンツールで自由な線を描画。Shift押下で直線制約
- **自由テキスト** — 部屋に属さない独立したテキストを任意の位置に配置
- **無限キャンバス** — ズーム・パン対応。マウスホイールやキーボードショートカットで自在に移動
- **複数選択** — Shift+クリックやエリア選択で複数部屋をまとめて移動・削除
- **Undo** — Ctrl+Z / ⌘Z で操作を元に戻す（最大50件）
- **マルチプロジェクト** — タブで複数プロジェクトを同時に開いて切り替え
- **自動保存** — localStorageに自動保存。ブラウザを閉じてもデータが残る
- **エクスポート** — PNG画像 / JSONファイルとして出力。JSONからの読み込みも可能

## 使い方

| 操作 | 動作 |
|------|------|
| 空白をドラッグ | 新しい部屋を作成 |
| 部屋をクリック | 選択 |
| Shift+クリック | 複数選択に追加/解除 |
| 選択中の部屋をドラッグ | 移動 |
| ハンドルをドラッグ | リサイズ |
| ダブルクリック | 部屋名の編集 |
| 右クリック | コンテキストメニュー（ドア・窓の追加など） |
| Delete / Backspace | 選択中の要素を削除 |
| Ctrl+Z / ⌘Z | 元に戻す |
| ホイール or ⌘+/⌘- | ズーム |
| Shift+ドラッグ | グリッドスナップを無効化（サブグリッド配置） |

## 開発

```bash
# 依存関係のインストール
npm install

# 開発サーバー起動 (localhost:5555)
npm run dev

# ビルド
npm run build

# テスト
npm test

# 型チェック / Lint / フォーマット
npm run typecheck
npm run lint
npm run format
```
