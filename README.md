# yakata-editor（館エディタ）

グリッドベースの間取り図エディタ。ミステリ小説の見取り図のように、黒い直線だけで館の構造を描くことに特化したWebアプリケーション。

## コンセプト

- 「部屋と廊下の構造を考えるためのツール」であり、インテリアデザインツールではない
- リサイズ可能な四角形を配置していくだけで間取りが完成する
- グリッドスナップにより、すべての線が水平・垂直に揃う
- ブラウザで動作し、インストール不要

## 現在の状態

`prototype.html` — 単一HTMLファイルで動作するプロトタイプ。基本機能は実装済み。
